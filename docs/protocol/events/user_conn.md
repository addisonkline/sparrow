# Event `user_conn`

## Schema
```json
{
    "head": {
        "protocol": "sparrow",
        "version": server version number as string,
        "origin": server URI as string,
        "timestamp": ISO 8601 timestamp as string,
        "type": "event/user_conn"
    },
    "body": {
        "user_id": connected user ID as string
    }
}
```
